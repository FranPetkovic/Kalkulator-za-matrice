SVOJSTVA ZBRAJANJA MATRICA

A + B = B + A - komunitativnost

(A + B) + C = A + (B + C) - asocijativnost

A + 0 = 0 + A = 0 - nulmatrica je neutralni element za zbrajanje

A * I = I * A = A - jedinica matrica je neutralni element za množenje
